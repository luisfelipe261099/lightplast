## 💻 Stack Tecnológico

- Backend: Node.js + TypeScript + Express
- Database: TiDB Cloud (MySQL-compatible)
- Frontend: HTML5 + Vanilla JS
- Deploy: Vercel (serverless)
- Automações: GitHub Actions (free)

## 🎯 Casos de Uso

1. **Review de Satisfação** - Contato automático 30 dias após pedido
2. **Resgate de Leads** - Última chance para leads frios (7 dias)
3. **Win-back Clientes** - Re-engajamento de clientes inativos (45 dias)
4. **Escalação Automática** - Notificação para leads muito qualificados
5. **Venda Cruzada** - Oferecer produtos complementares

## 📊 Dashboard Incluído

Uma interface visual (HTML/CSS/JS) que mostra:
- KPIs em tempo real
- Lista de clientes com histórico
- Leads com score automático
- Agenda de retornos
- Pedidos e receita
