## 🚀 COMO USAR O SISTEMA

### PASSO 1: Verificar Pré-requisitos
```bash
# Node.js instalado?
node --version
# npm instalado?
npm --version
```

### PASSO 2: Instalar Dependências (se novo)
```bash
npm install
```

### PASSO 3: Configurar Banco de Dados
```
1. Obter credenciais TiDB Cloud
2. Criar arquivo .env com:
   TIDB_HOST=seu_host
   TIDB_USER=seu_user
   TIDB_PASSWORD=sua_senha
   TIDB_DATABASE=seu_db

3. Executar setup.sql no database
4. (Opcional) Importar DADOS_FICTICIOS.sql
```

### PASSO 4: Iniciar Servidor
```bash
npm start
```

### PASSO 5: Abrir no Navegador
```
http://localhost:3000
```

### PASSO 6: Testar Cada Módulo
```
✅ Dashboard → Veja KPIs e gráficos

✅ Leads → Crie um novo lead
✅ Orçamentos → Gere orçamento
✅ Pedidos → Registre um pedido

```
