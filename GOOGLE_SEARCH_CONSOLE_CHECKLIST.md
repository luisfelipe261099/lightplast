# Google Search Console Checklist - LightPlast

## 1) Publicar e validar URLs principais
- Verificar homepage: https://lightplast.vercel.app/
- Verificar catalogo: https://lightplast.vercel.app/catalogo.html
- Verificar blog: https://lightplast.vercel.app/blog.html
- Verificar categorias e produtos novos

## 2) Cadastrar propriedade no Google Search Console
- Entrar em https://search.google.com/search-console
- Adicionar propriedade de dominio (recomendado) ou prefixo de URL
- Concluir verificacao DNS

## 3) Enviar sitemap
- No menu "Sitemaps", enviar:
- https://lightplast.vercel.app/sitemap.xml
- Confirmar status como "Sucesso"

## 4) Solicitar indexacao das paginas estrategicas
- Ferramenta "Inspecao de URL"
- Solicitar indexacao para:
  - homepage
  - catalogo
  - categorias
  - produtos principais
  - posts de blog

## 5) Monitorar cobertura de indexacao
- Ver relatorio "Paginas"
- Corrigir URLs "Descoberta - atualmente nao indexada"
- Corrigir URLs "Rastreada - atualmente nao indexada"

## 6) Monitorar performance (CTR e posicao)
- Relatorio "Resultados da pesquisa"
- Filtrar por pagina para identificar quais URLs convertem melhor
- Priorizar expansao de conteudo das paginas com mais impressoes e baixa CTR

## 7) Melhorar snippets para subir CTR
- Revisar title e description de paginas com alta impressao e baixa taxa de clique
- Garantir title direto com palavra-chave principal
- Garantir description orientada a beneficio + CTA

## 8) Ritmo de crescimento recomendado
- Publicar 2 novos artigos por mes
- Publicar 2 novas paginas de produto por mes
- Atualizar sitemap apos cada lote

## 9) Checklist mensal de SEO tecnico
- Confirmar status do sitemap
- Revisar erros de cobertura
- Revisar Core Web Vitals
- Revisar links internos entre catalogo, categorias, produtos e blog
