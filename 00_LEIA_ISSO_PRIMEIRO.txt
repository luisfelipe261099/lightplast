🚀 COMO COMEÇAR (5 MINUTOS)
════════════════════════════════════════════════════════════════════════════════

PASSO 1: Abra PowerShell/CMD em c:\xampp\htdocs\1

PASSO 2: Instale dependências
    npm install

PASSO 3: Configure .env com credenciais TiDB Cloud
    TIDB_HOST=seu_host.tidbcloud.com
    TIDB_USER=seu_usuario
    TIDB_PASSWORD=sua_senha
    TIDB_DATABASE=seu_database

PASSO 4: Inicie o servidor
    npm start

PASSO 5: Abra no navegador
    http://localhost:3000

PASSO 6: Teste cada módulo!
